# PatioTime Demo 3 — Page Topology

Target: `https://patiotime.loftocean.com/demo3/`

1. Header overlay — absolute at the hero, `z-index: 999`; desktop contact strip plus 101px navigation row, mobile 80px logo/menu/language row. Sticky replacement appears after scrolling.
2. Hero — full viewport on desktop (702px in captured browser), 640px mobile; centered logo/header overlap, two-line Cormorant title, orange PatioTime word, caption and bottom opening-hours label.
3. Story — dark `#0e0d0a`; desktop three columns with 360×480 images and centered copy, mobile two images above stacked text.
4. Experience — full-bleed parallax photography with a transparent handwritten PNG centered above it.
5. Cocktail gallery — horizontal, time/click driven carousel; four visible desktop cards, one mobile card; variable portrait ratios and dot navigation.
6. Menu banner — full-bleed parallax image, centered eyebrow and “À la Carte”.
7. Menu cards — three portrait image CTAs on desktop and vertically stacked on mobile; image overlay text, hover zoom/darken.
8. Events — large dark introduction followed by a single-image slider with next/previous arrows and dots. One 768×512 slide centered on desktop, 328×219 mobile.
9. Footer — full-bleed background photo/overlay with centered venue details, orange maps button, social icons and copyright.
10. Global overlays — right-side navigation panel and full-screen reservation modal.

The page uses ordinary document scrolling with scroll-triggered fade-up entrances and faux parallax image movement. It does not use scroll snap.
